# deoplete-zsh

Zsh completion for deoplete.nvim


## Required

### Neovim
https://github.com/neovim/neovim/

### deoplete.nvim
https://github.com/Shougo/deoplete.nvim


### zsh/zpty module

```zsh
zmodload zsh/zpty
```


## Configuration

It saves compdump file in `$DEOPLETE_ZSH_CACHE_DIR` or `$XDG_CACHE_HOME` or
`$HOME/cache` directory.
